# SQL-Server-Management-Studio

A collection of reusable SQL query files for SQL Server Management Studio (SSMS).

## Files

| File | Description |
|------|-------------|
| `01_database_operations.sql` | Create, alter, rename, and drop databases; check database properties and file sizes |
| `02_table_operations.sql` | Create, alter, rename, and drop tables; add/remove columns and constraints; temp tables |
| `03_data_manipulation.sql` | INSERT, SELECT, UPDATE, DELETE, MERGE; JOINs, CTEs, window functions, transactions |
| `04_stored_procedures.sql` | Create stored procedures with input/output parameters and error handling |
| `05_views.sql` | Create regular and indexed (materialized) views |
| `06_indexes.sql` | Create, rebuild, reorganize, and drop indexes; find missing and unused indexes |
| `07_user_management.sql` | Create logins and users; grant, revoke, and deny permissions; manage roles |
| `08_database_maintenance.sql` | Backup and restore databases; DBCC checks; index rebuild; kill connections |
| `09_query_performance.sql` | Monitor running queries, blocking, wait stats, index usage, and memory |
| `10_triggers.sql` | Create DML and DDL triggers for auditing, soft-delete, and business rules |

## Usage

Open any `.sql` file in SQL Server Management Studio, update the database name and file paths as needed, and execute the relevant sections.
